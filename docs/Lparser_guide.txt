PREREQUISITES

    
Python3 (version 3.4 or higher)




INSTALL

Unzip the archive and go to the folder src/





EXECUTE


The command line syntax is:



python main.py [l_program]



For example, run

   


>python main.py examples/t1.l

You should see

[['rule', ['sent', ['unit', ['literal', ['atom', ('identifier', 'a')]]]]], ['rul
e', ['sent', ['unit', ['literal', ['atom', ('identifier', 'b')]]]], ['sent', ['u
nit', ['literal', ['atom', ('identifier', 'a')]]]]]]

The trees are defined in: 
https://github.com/iensen/genparser/raw/master/docs/main/astgen.pdf
